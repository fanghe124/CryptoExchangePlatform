System Requirements

System Requirements

 

  • PHP 5.5+
  • MYSQL 5.1+
  • mod_rewrite Apache
  • Mcrypt Extension
  • MBString Extension
  • GD Extension
  • MYSQLi Extension
  • PDO Extension
  • OpenSSL Extension
  • CURL Extension

In most hosting accounts these extensions are enabled by default. But you should consult with your hosting provider.

Installation

Configure database connection

You need to have previously setup database from the cPanel.

Here is a good tutorial how to setup MySQL database in cPanel if you are not familiar with this.

  1. Login to cPanel by accessing www.yourdomain.com/cpanel and navigate go MySQL Databases.
  2. Create database.
  3. Create user and setup your password.
  4. Add the user to database by selecting the database and the username.

Make sure you have checked All privileges when adding the user to database.

 

Upload Files & Install

Upload the files to your hosting folder and navigate to the location where you will install CryptoEra  http://www.domain.com/installer

 

 

 

Make sure all file and folder permissions are met and click the continue button

Now provide your Database details and create your base admin account as show in the image below

Create your base admin account , select your timezone and click the install and finish button as show below

 

 

Setup Cron Job

CryptoEra needs to periodically update cryptocurrency market data from various internet sources.
Cron task should run once per minute. Complete cron task entry should be similar to:

* * * * * wget --quiet -O /dev/null http://www.example.com/cron/run

Don't forget to replace http://www.example.com/ with the actual URL of your app.

Access Admin

CryptoEra comes with an extensive admin panel to manage your site

You can access the admin after login via http://www.example.com/admincp

Make sure you replace the example.com with your own domain

How To

This section explain various usage steps for CryptoEra platform

How to edit coins

To edit any coin , go to your admincp -> Coins Manager -> search for the coin -> click the edit and save

How to feature coins

You can do that with the instruction giving in the how to edit coins

How to hide coins

You can hide coins by going to your admincp -> settings -> general ->hide coins

You need to put the symbol of the coins you want to hide separated by comma

How to customize look

You can easily customize the look to your taste by going to admincp -> appearance

You can also add custom css content in the custom css field

How to add Google ads

You can add your google ads code in your admincp -> Manage ads

There are two possible location in different pages of your CryptoEra site

1. Top Location - Display ads on top of every page
2. Side location - Display ads on side bar

How to edit Pages content

Editing pages content like terms, about e.t.c is easy as just going to your admincp -> Manage pages

How to add Google Analytic Code

You can add your google analytics code by going to your admincp -> settings -> api

                        RewriteCond %{REQUEST_FILENAME} !-d

Upgrade Instructions
